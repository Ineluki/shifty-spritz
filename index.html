Shifty Spritz by Andrew Milson

Shifty Spritz is a FREE Spritz-type speed reader extension for browsers. It will save you HEAPS of time! Try it out on this page by selecting some text and holding shift

Download its FREE and you won't regret it!

  • Chrome Web Store

About Shifty Spritz

Shifty Spritz is an extremely crude implementation of Spritz. I made it because I struggle reading fast. I found that the Spritz reader was perfect for me as I was able to read fast and with great comprehension. Although I found that Spritz and similar readers where to clunky and slow, so I decided to spin up my own. Shifty Spritz speed reader is open source and you can view all the code on GitHub. I hope you love using it as much as I do!

Features

+ WPM selector
+ Spritz-type speed reading
+ Reads selected text
